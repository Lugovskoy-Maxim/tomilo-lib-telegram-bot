# Telegram Bot для уведомлений о новых главах

Этот бот уведомляет пользователей о появлении новых глав манги через Telegram.

## Установка

1. Клонируйте репозиторий:
   ```
   git clone <repo-url>
   cd tomilo-lib-telegram-bot
   ```

2. Установите зависимости:
   ```
   npm install
   ```

3. Создайте файл `.env` на основе `.env.example` и укажите ваш токен бота:
   ```
   BOT_TOKEN=ваш_токен_бота
   API_BASE_URL=http://localhost:3000/api
   ```

## Запуск

```
npm start
```

Для разработки с автоматической перезагрузкой:
```
npm run dev
```

## Решение проблем

Если вы видите предупреждение `DeprecationWarning: The 'punycode' module is deprecated`, это связано с внутренними зависимостями библиотеки telegraf. Предупреждение подавляется при запуске через `npm start`.

Если сообщения о запуске бота не отображаются сразу в консоли, это связано с буферизацией вывода. Сообщения будут отображены после завершения процесса (Ctrl+C). Это нормальное поведение Node.js в некоторых средах.

## Использование

1. Найдите вашего бота в Telegram
2. Напишите команду `/start`
3. Бот будет уведомлять вас о новых главах каждые 30 минут

## Функциональность

- Уведомления о новых главах
- Ссылки для чтения глав
- Автоматическая проверка каждые 30 минут
